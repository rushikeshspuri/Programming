# Programming

A simple collection of my programming practice files while learning logic building and core programming concepts.

This repository contains programs written in **C**, **C++**, **Java**, and **JavaScript**. The main goal is to improve problem-solving, understand syntax clearly, and build strong programming logic step by step.

## Repository Overview

| Folder | What it contains |
| --- | --- |
| `C_programming` | C programs for basics, logic building, loops, functions, arrays, strings, and problem solving |
| `C++_Programming` | C++ practice programs and object-oriented programming basics |
| `java_programming` | Java programs for core concepts, classes, objects, and logical practice |
| `Java_Script_Programming` | JavaScript practice files for basic web/programming concepts |

## Learning Focus

- Building logic before writing code
- Practicing algorithms and dry runs
- Understanding input, processing, and output
- Solving programs using step-by-step thinking
- Improving consistency through regular coding practice

## How I Study Programs

For each program, I try to understand:

```text
Problem
Algorithm
Variables used
Dry run
Code logic
Mistakes to avoid
```

## Goal

To become stronger at programming by writing code, making mistakes, correcting them, and learning how to think like a software developer.
